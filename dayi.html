答疑 app接口 文档

注册：apps/member/aq_sign_up
email 必填
name 必填
password 必填
phone_os // 'ios' or 'android'
返回值：
{
response_status: "ok",
response_data:
{
"code":"1",
"msg":"注册成功"
}
}
response_data 其它返回值：
{"code":"-1","msg":"email格式不正确"}
{"code":"-2","msg":"该手机已被注册过"}
{"code":"-3","msg":"密码长度应为 6-20 位字符"}
{"code":"-4","msg":"手机验证码不正确"}
{"code":"-5","msg":"注册失败，服务忙"}
